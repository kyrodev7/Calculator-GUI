Calculator GUI (Tkinter)

A simple calculator built using Python and Tkinter.  
It supports basic operations like addition, subtraction, multiplication, division, and exponentiation.

Features
- Simple graphical user interface
- Buttons for each operation
- Clear button to reset inputs
- Displays results instantly

What I Learned
- How to use Tkinter for GUI development
- How to handle user input
- How to update UI elements dynamically
- How to structure a small Python project

How to Run
1. Make sure you have Python installed.
2. Run the script with:

   python main.py
